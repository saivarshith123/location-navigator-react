# 🧭 Location Navigator React App

A simple and responsive React application that provides one-click navigation to important locations using Google Maps.

## 🚀 Features

* One-click Google Maps navigation
* Light Mode / Dark Mode toggle
* Responsive design for mobile and desktop
* Clean and modern user interface
* Fast and lightweight React application
* Easy customization of locations and map links

## 📱 Locations Included

* 🏠 PG Accommodation
* 📚 Coaching Center
* 📖 Study Room

## 🛠️ Built With

* React
* Vite
* CSS3
* Google Maps

## 📂 Project Structure

```text
src/
├── App.jsx
├── App.css
├── main.jsx
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/location-navigator-react.git
```

Navigate to the project folder:

```bash
cd location-navigator-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## 🗺️ Adding Your Locations

Inside `App.jsx`, replace the placeholder links:

```jsx
mapLink: "PASTE_PG_LOCATION_LINK_HERE"
```

with your actual Google Maps links.

Example:

```jsx
mapLink: "https://maps.app.goo.gl/example"
```

## 🌙 Dark Mode

Use the toggle switch in the top-right corner to switch between Light Mode and Dark Mode.
